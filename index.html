Display & Position


The position property allows you to specify the position of an element and the display property allows you to control how an element flows vertically and horizontally in a document. Here are their most common usages:

Display

Property Name Description
display: inline Inline Displays an element as an inline element. Any height and width properties will have no effect.
display: block Block Displays an element as a block element It starts on a new line, and takes up the whole width
display: inline-block Inline-Block Displays an element as an inline-level block container. The element itself is formatted as an inline element, but you can apply height and width values

Position

Property Name Description
position: relative Relative When set to relative, an elements position is relative to its default position on the page.
position: fixed Fixed When set to fixed, an element's position can be pinned to any part of the web page. The element will remain in view no matter what.
position: static Static When set to fixed, an element's position can be pinned to any part of the web page. The element will remain in view no matter what.
position: sticky Sticky When set to sticky, an element can stick to a defined offset position when the user scrolls its parent container.
position: absolute Absolute When set to absolute, an element's position is relative to its closest positioned parent element. It can be pinned to any part of the web page, but the element will still move with the rest of the document when the page is scrolled.
